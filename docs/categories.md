# Categories

Learn how to work with icon categories in @pelatform/icons.

## 📋 Overview

The library includes 42 categories that organize 5000+ icons into logical groups. Each category is automatically generated by scanning the `icons/` folder, making it easy to add new categories.

## 🎯 Category Structure

Each category has the following structure:

```typescript
interface Category {
  id: string; // kebab-case identifier, e.g., "animals", "arrows"
  title: string; // Title Case display name, e.g., "Animals", "Arrows"
}
```

## 💡 Usage Examples

### 1. Category Filter Dropdown

Create a category selector for your icon picker:

```tsx
import { useState } from 'react';
import { categories, type Category } from '@pelatform/icons/categories';

function CategoryFilter() {
  const [selectedCategory, setSelectedCategory] = useState<string>('all');

  return (
    <select
      value={selectedCategory}
      onChange={(e) => setSelectedCategory(e.target.value)}
      className="category-select"
    >
      {categories.map((category: Category) => (
        <option key={category.id} value={category.id}>
          {category.title}
        </option>
      ))}
    </select>
  );
}
```

### 2. Category Tabs

Create clickable category tabs:

```tsx
import { useState } from 'react';
import { categories, type Category } from '@pelatform/icons/categories';

function CategoryTabs() {
  const [activeCategory, setActiveCategory] = useState<string>('all');

  return (
    <div className="category-tabs">
      {categories.map((category: Category) => (
        <button
          key={category.id}
          onClick={() => setActiveCategory(category.id)}
          className={`tab ${activeCategory === category.id ? 'active' : ''}`}
        >
          {category.title}
        </button>
      ))}
    </div>
  );
}
```

### 3. Type-Safe Category Selection

Use TypeScript for type-safe category handling:

```tsx
import { categories, type Category } from '@pelatform/icons/categories';
import type { IconCategory } from '@pelatform/icons/types';

function filterByCategory(iconList: string[], categoryId: IconCategory) {
  if (categoryId === 'all') return iconList;

  // Use icon-category.json for filtering
  const iconCategoryMap = require('@pelatform/icons/out/icon-category.json');

  return iconList.filter((iconName) => {
    const category = iconCategoryMap[iconName]?.category.toLowerCase();
    return category === categoryId;
  });
}

// Usage
const healthIcons = filterByCategory(allIcons, 'health');
const animalIcons = filterByCategory(allIcons, 'animals');
```

### 4. Category Statistics

Display icon count per category:

```tsx
import { categories } from '@pelatform/icons/categories';
import iconCategoryMap from '@pelatform/icons/out/icon-category.json';

function CategoryStats() {
  const stats = categories.map((category) => {
    if (category.id === 'all') {
      return {
        ...category,
        count: Object.keys(iconCategoryMap).length,
      };
    }

    const count = Object.values(iconCategoryMap).filter(
      (icon) => icon.category.toLowerCase() === category.id,
    ).length;

    return { ...category, count };
  });

  return (
    <div>
      {stats.map((category) => (
        <div key={category.id}>
          {category.title}: {category.count} icons
        </div>
      ))}
    </div>
  );
}
```

### 5. Breadcrumb Navigation

Create breadcrumb navigation with categories:

```tsx
import { categories, type Category } from '@pelatform/icons/categories';

function Breadcrumb({ categoryId }: { categoryId: string }) {
  const category = categories.find((cat) => cat.id === categoryId);

  return (
    <nav className="breadcrumb">
      <span>Icons</span>
      <span>/</span>
      {category && <span>{category.title}</span>}
    </nav>
  );
}
```

### 6. Icon Grid by Category

Display icons grouped by category:

```tsx
import { categories, type Category } from '@pelatform/icons/categories';
import iconList from '@pelatform/icons/icon-list';
import iconCategoryMap from '@pelatform/icons/out/icon-category.json';

function IconGrid() {
  const getIconsByCategory = (categoryId: string) => {
    if (categoryId === 'all') return iconList;

    return iconList.filter((iconName) => {
      const category = iconCategoryMap[iconName]?.category.toLowerCase();
      return category === categoryId;
    });
  };

  return (
    <div>
      {categories.map((category: Category) => {
        const icons = getIconsByCategory(category.id);

        return (
          <section key={category.id}>
            <h2>
              {category.title} ({icons.length})
            </h2>
            <div className="icon-grid">
              {icons.map((iconName) => (
                <IconCard key={iconName} name={iconName} />
              ))}
            </div>
          </section>
        );
      })}
    </div>
  );
}
```

## 🔧 Advanced Usage

### Custom Category Mapping

Create your own category mapping:

```tsx
import { categories } from '@pelatform/icons/categories';
import type { IconCategory } from '@pelatform/icons/types';

const categoryColors: Record<IconCategory, string> = {
  all: '#6366f1',
  animals: '#22c55e',
  arrows: '#3b82f6',
  health: '#ef4444',
  // ... etc
};

function CategoryBadge({ categoryId }: { categoryId: IconCategory }) {
  const category = categories.find((cat) => cat.id === categoryId);
  const color = categoryColors[categoryId];

  return (
    <span className="category-badge" style={{ backgroundColor: color }}>
      {category?.title}
    </span>
  );
}
```

### Category Search

Search within categories:

```tsx
import { categories, type Category } from '@pelatform/icons/categories';
import iconList from '@pelatform/icons/icon-list';
import iconCategoryMap from '@pelatform/icons/out/icon-category.json';

function searchIcons(query: string, categoryId: string) {
  const categoryIcons =
    categoryId === 'all'
      ? iconList
      : iconList.filter((iconName) => {
          const category = iconCategoryMap[iconName]?.category.toLowerCase();
          return category === categoryId;
        });

  return categoryIcons.filter((iconName) =>
    iconName.toLowerCase().includes(query.toLowerCase()),
  );
}
```

## 📝 Tips

1. **Use "all" category** for showing all icons
2. **Category IDs are kebab-case** (e.g., "version-control")
3. **Category titles are Title Case** (e.g., "Version Control")
4. **Auto-generated** - Categories are created from folder names in `icons/`
5. **Type-safe** - Use `IconCategory` type for compile-time safety
